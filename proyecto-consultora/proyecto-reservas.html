Proyecto de Reservas

Propuesta de solución para gestionar reservas, disponibilidad y confirmaciones de forma simple y escalable.

Objetivo

Crear una plataforma web donde el usuario pueda solicitar una reserva, y el equipo pueda gestionar disponibilidad y confirmaciones.

Funcionalidades

  • Formulario de solicitud de reserva.
  • Validación de datos y confirmación por email (simulada).
  • Panel de control básico para revisar solicitudes.

Tecnologías

HTML semántico, CSS con variables y Flexbox, y diseño responsive.

Ficha rápida

  • Estado: Prototipo
  • Versión: 1.0
  • Entrega: FP DAM
